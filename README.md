# HolyGrail-OG

Infinitely respawning anvils and end crystals at spawn and in duels arenas. A cup that never runs dry. Built for [TrueOG](https://true-og.net/).

Gradle Version: 8.1.1

Java Version: 17

Minecraft API: 1.18.2
